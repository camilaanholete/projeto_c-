# Projeto de Aulas C#

Este repositório contém exercícios e exemplos desenvolvidos durante o aprendizado da linguagem C#, organizados por aulas. O foco é praticar conceitos fundamentais de programação com C#, desde o básico até tópicos mais avançados como orientação a objetos e acesso a banco de dados.

## 📚 Conteúdo

Cada pasta representa uma aula ou módulo:

- `Aula03/` – Estruturas de decisão e repetição
- `Aula04/` – Métodos e passagem de parâmetros
- `Aula05/` – Arrays e listas
- `Aula06/` – Classes, objetos e encapsulamento
- `Aula07/` – Leitura e escrita de arquivos
- `Aula08/` – Conexão com banco de dados (ADO.NET)
- `Aula_CRUD/` – Sistema CRUD completo

> **Nota:** Os nomes das pastas podem variar. Consulte a estrutura real do repositório para ajustes.

## 🚀 Como Executar

### Pré-requisitos

- [.NET SDK](https://dotnet.microsoft.com/download)
- [Visual Studio](https://visualstudio.microsoft.com/) ou outro editor com suporte a C#

### Passos

1. Clone o repositório:
   ```bash
   git clone https://github.com/camilaanholete/projeto_c-.git
   cd projeto_c-
````

2. Abra a pasta da aula desejada no Visual Studio.

3. Compile e execute o projeto (geralmente usando `F5` ou via terminal com `dotnet run`).

## 🛠 Tecnologias Utilizadas

* C#
* .NET Framework ou .NET Core (dependendo da aula)
* Visual Studio

## 💡 Aprendizados

Este projeto cobre:

* Estruturas de controle (`if`, `while`, `for`)
* Métodos, escopo de variáveis e passagem por valor/referência
* Programação orientada a objetos
* Manipulação de arquivos
* Conexão e operações com banco de dados
